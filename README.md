## 🎯 What is LezerPrint?

LezerPrint is a **professional-grade 3D printer management system** that combines cutting-edge web technologies with practical 3D printing solutions. Built as both a **functional tool** and a **portfolio showcase**, it demonstrates advanced full-stack development capabilities while solving real-world 3D printing challenges.

### 🌟 Why LezerPrint?

- **Portfolio Showcase** - Demonstrates modern full-stack development skills
- **Real-World Application** - Solves actual 3D printing workflow challenges  
- **Production Ready** - Built with enterprise-grade security and scalability
- **Open Source** - MIT licensed for learning and contribution

## 🚀 Quick Start

Get LezerPrint running in under 5 minutes:

```bash
# 1. Clone the repository
git clone https://github.com/Avi-Lezerovich/LezerPrint.git
cd LezerPrint

# 2. Install dependencies
npm run setup

# 3. Start all services
./start.sh

# 4. Open your browser
open http://localhost:3000
```

**🎉 That's it!** LezerPrint will be running with:
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:3001  
- **Database Admin**: http://localhost:5555

## 📸 Features

### 🖨️ **Core Printer Management**
- Real-time printer control and monitoring
- Complete G-code command support
- Multi-format file uploads (STL, G-code, OBJ, 3MF)
- Intelligent print queue management

### 📊 **Advanced Analytics**
- Performance metrics and success rate tracking
- Material usage and cost analysis
- Historical data with trend visualization
- Failure analysis with improvement recommendations

### 🎥 **Live Monitoring**
- Real-time camera streaming
- Temperature and progress tracking
- Automatic timelapse creation
- Emergency stop and safety controls

### 🔧 **Professional Features**
- Multi-user support with role-based access
- Custom print profiles and settings
- G-code terminal with macro support
- Comprehensive audit logging
